# 最小命令行聊天助手（Python）

这是一个最小可运行的 Python 命令行聊天助手示例。

## 项目结构

```text
.
├── README.md
├── requirements.txt
└── main.py
```

## 安装

1. （可选）创建并激活虚拟环境：

```bash
python -m venv .venv
source .venv/bin/activate
```

2. 安装依赖：

```bash
pip install -r requirements.txt
```

## 运行

```bash
python main.py
```

## 使用方式

- 启动后输入任意内容，助手会返回一个最简单的回显回复。
- 输入 `exit`、`quit` 或 `q` 可退出程序。
